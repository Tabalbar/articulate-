# Articulate

# Requirements

- Must have Node version v14.16.0+ installed

# Installation

### Install dependencies

```console 
foo@bar:~$ cd Articulate-
```

```console 
foo@bar:~$ yarn
```

```console 
foo@bar:~$ cd client
```

```console 
foo@bar:~$ yarn
```

## How to start

```console 
foo@bar:~$ cd Articulate-
```

```console 
foo@bar:~$ yarn dev
```

## Covid Data

In the project directory, there is a file named "Covid-19.csv." 
This will be the test dataset to load into Articulate.
