# Articulate

# Requirements

- Must have Node version v14.16.0+ installed

# Installation

### Install dependencies

```console 
foo@bar:~$cd Articulate-
```

`yarn`

`cd client`

`yarn`

## How to start

`cd Articulate-`

`yarn dev`

## Covid Data

In the project directory, there is a file named "Covid-19.csv"
